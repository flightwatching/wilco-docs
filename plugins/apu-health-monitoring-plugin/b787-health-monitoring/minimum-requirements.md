# Minimum requirements

## Data from the A/C

OOOI, APU reports, CMS reports, etc...&#x20;

## Data from the ground infrastructure

none

## Plugins list

To enable health monitoring on a server, you need to install:

On A320 fleet (both APUs: APS3200 & GTCP131-9A)

* fw.acars.core
* fw.a320
* fw.acars.oooi
* fw.adsb
* fw.a320.hms
* fw.hms



On B787 fleet:

* fw.acars.core
* fw.acars.oooi
* fw.adsb
* fw.b787
* fw.b787.hms
* fw.hms
